What is a stack?

A stack is a linear data structure that applies the Last-In-First-Out(LIFO) principle. This means that when inserting and deleting data, the data is inserted at the top of the structure and also deleted from the top of the structure. This is similar to how we stack toy blocks. When adding a new block it is added on top of the last block. Also when removing blocks we have to start by taking away the block at the top.






In the example of the building blocks above we see that a new block is pushed on top of the last one. Also, when removing a block from the stack one must first start by removing the block at the top. The stack ADT works the same way. Inserting into a stack is done by a function called 'push(value)' and deleting from the stack is done by a function called 'pop()'. this is shown in the example below:






The stack ADT has 3 main funtions:
   1. push(value) - inserting data into the stack.
   2. pop() - deleting data from the stack.
   3. display() - displaying the contents of the stack.


The steps to implementing the push function are:
   1. Check if the stack is full. (only done when using arrays)
   2. If the stack is full, produces an error and exit.
   3. If the stack is not full, increments top to point next empty space.
   4. Inserts data element to the stack location, where top is pointing.

The steps to implementing the pop function are:
   1. Check if the stack is empty.
   2. If the stack is empty, produces an error and exit.
   3. If the stack is not empty, deletes the value at the location that top is pointing to.
   4. decrements top to point to previous value


A stack can be implemented 2 ways:
   1. Using Arrays
   2. Using Linked-Lists
